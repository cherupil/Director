Let’s Go!

I’ve finally added support for animating DOM properties, so we’re no longer limited to animating JavaScript objects.

Next I’ve added support for animating CSS Transforms. Right now just 2D transforms, but later down the line I’ll add 3D transform support.

Lastly, I’ve made CSS class toggling a part of the library, so if you want to animate purely through adding and removing classes, you can do that too.

Oh and one more thing… you can implement this cool hiding technique with a little bit of CSS and the added pinning offset capability.
